# debloated-oneui1-stock-galaxy-s8(project-nightmare)
 one ui stock for the galaxy s8
IM NOT RESPONSIBLE FOR ANY BRICKED DEVICES!

this is a debloated and deknoxed one ui 1 rom based on the A2N kernel from the telegram user @ace2nutzer

# HOW TO INSTALL
boot to twrp (or your similar custom recovery)
format data and do a full wipe on the system (cache, dalvik, system, data, internal storage, etc.)
after flash the rom like any other.
reboot and enjoy
